ADREST
######

Adrest is Another Django REST. Django application for simple make HTTP REST API.

Documentation in construction.

Requirements
=============

- python >= 2.5
- django >= 1.2

Installation
=============

**Adrest** should be installed using pip: ::

    pip install adrest
